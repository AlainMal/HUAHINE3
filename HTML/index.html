Guide d'utilisation - HUAHINE

Introduction

Bienvenue dans le guide d'utilisation de HUAHINE. Cette application vous permet de gérer et visualiser vos données géographiques.

Fonctionnalités principales

  • Visualisation des cartes avec votre direction et votre parcours, ainsi que les autres bateaux sur AIS.
  • Importer les données du bus CAN sur la table
  • Exporter le résultat en CSV sur NMEA 2000

Comment utiliser l'application

  1. Lancez l'application
  2. Raccordez l'adaptateur sur un port USB
  3. Raccordez sur le bus CAN le H et L
  4. Cliquez sur le bouton "OPEN"
  5. Cliquez sur le bouton "Lecture du bus CAN"
  6. Vous pouvez visualiser votre position sur la carte.
  7. Vous voyez aussi votre projection ainsi que les autres bateaux sur AIS

Enregister et importer

  1. Cliquez sur le bouton ouvrir le fichier texte
  2. Enregistrez le bus CAN en cochant la case à cocher
  3. Vous vous arrêtez quand vous voulez
  4. Cliquez sur le bouton "Importer le fichier CAN"
  5. Vous pouvez voir sur la table les différentes lignes importées
  6. Cliquez sur la table et vous voyez le résultat en NMEA 2000 sur la gauche

Exporter le NMEA 2000

  1. Cliquez sur le bouton "Exporter en NMEA 2000"
  2. Choisissez votre fichier au format .csv
  3. Enregistrez, puis vous validez ou choisissez d'autres lignes à exporter
  4. Vous pouvez voir le fichier en .csv en cliquant sur "Yes"
Astuce : Utilisez la touche F1 à tout moment pour accéder à l'aide.
