Importer les données du bus CAN

Comment importer les données du bus CAN

  1. Cliquez sur "Ouvrir un fichier texte" dans la barre d'outils.

      Choisissez un fichier éxistant.

      Cliquer sur 'Enregistrer'.

  2. Cliquez sur importer le "fichier texte CAN dans la table" dans la barre d'outils.

      Vous avez la possibilité de valider, ou de choisir les numéros des lignes.

  3. Vous avez à présent la possibilité de cliquer sur la table.

      Vous verrez alors les résultats de la conversion en NMEA 2000.

Remarque : Si vous créez un nouveau fichier, il ne se passera rien, il faudra d'abord l'enregistrer.
  • Regardez un exemple du tableau rempli.
  • Vous voyez sur la partie gauche le résultat traduit en NMEA 2000.
